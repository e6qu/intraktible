Agentic decision platform

Decisions, as data.

An open-source, self-hosted decision platform: versioned flows and AI agents on an event-sourced, fully auditable core — with four-eyes change control and built-in model-risk governance (SR 11-7 / SS1/23). A persona-adaptive UI over one API.

Build decisions visually

Compose versioned flows from rules, decision tables, scorecards, predictive models, and AI agents on a canvas. Test before you publish.

Event-sourced & replayable

Every decision is recorded as an immutable event with a full node-by-node trace — inspect, replay, and explain any outcome.

See a trace →

Governed by design

Four-eyes deployment approval, an immutable audit log, RBAC, and a model-risk inventory with drift monitoring — built in, not bolted on.

See the model-risk inventory →

Human-in-the-loop

When judgement is needed, a flow escalates to a case with SLA tracking; reviewers triage and resolve, and the trail links back to the decision.

Persona-adaptive UI

One API, many lenses: the same platform re-prioritises for a builder, an operator, a manager, a developer, or an executive.

Open source, self-hosted

AGPL-3.0, a Go modular monolith with a SvelteKit UI, run on your own infrastructure with no per-decision fees.

Build a flow→ Deploy→ Run decisions→ Escalate to review→ Govern
